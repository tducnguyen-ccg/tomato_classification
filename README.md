# Tomato classification
Tomato color classification to Red, Yellow and Brown type

## Dataset
Tomato image dataset (and other fruits) is available in this link:
https://github.com/Horea94/Fruit-Images-Dataset

## Requirements
* OpenCV
* Numpy
* Tensorflow
